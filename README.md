# SynkBrands

SynkBrands is a modern web application built with a fast and scalable frontend setup. This project uses a clean development workflow and is easy to run locally or deploy.

## Project Setup

### Prerequisites

* Node.js (recommended via nvm)
* npm

### Installation

```bash
# Clone the repository
git clone https://github.com/yashashree12-12/SynkBrands.git

# Navigate to the project folder
cd SynkBrands

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will start with hot-reloading enabled for faster development.

## Editing the Code

You can edit the project using:

* Any local IDE (VS Code recommended)
* GitHub web editor
* GitHub Codespaces

All changes can be committed directly to this repository.

## Tech Stack

* Vite
* TypeScript
* React
* shadcn/ui
* Tailwind CSS

## Deployment

Build the project using:

```bash
npm run build
```

You can deploy the generated build folder to any static hosting platform (Vercel, Netlify, etc.).
